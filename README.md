# IT Help Desk Portfolio

This repository contains hands-on labs and troubleshooting scenarios demonstrating foundational IT support skills across Windows, Active Directory, networking, and Linux environments.

The labs simulate real help desk tasks such as user login issues, network connectivity troubleshooting, DNS failures, and Active Directory administration.

- Windows operating system troubleshooting
- Active Directory user and group management

- Linux command-line navigation

- Privilege escalation using sudo
- Virtual machine deployment and management
- Structured IT troubleshooting documentation
