# Linux Lab – File System & Permissions

## Environment
- Ubuntu ARM running in UTM
- Apple Silicon macOS host

---

## Skills Practiced

### File Navigation
- Used pwd, ls, cd to navigate directories
- Created and removed files and folders

### Permissions
- Viewed file permissions using ls -l
- Modified permissions using chmod
- Demonstrated restricted access with 600 permission setting

### Privilege Escalation
- Attempted software installation without sudo (failed)
- Successfully installed htop using sudo apt install

---

## Simulated Ticket

User unable to install software due to insufficient privileges.

### Resolution:
Used sudo to elevate permissions and complete installation.

---

## Concepts Demonstrated
- Linux file structure
- Permission model (owner/group/others)
- Command-line troubleshooting
- Privilege escalation (sudo)
